Puck
====

Developed for the [Planets Project](http://www.planets-project.eu/) to find significant areas in digital images and save them by using the [extensible Characterization Language (XCL)](http://planetarium.hki.uni-koeln.de/planets_cms/index.php).

The approach is described in the paper [Guttenbrunner, Rauber, Thaller, Wieners (2010): "Same Same But Different – Comparing RenderingEnvironments for Interactive Digital Objects"](http://publik.tuwien.ac.at/files/PubDat_190888.pdf).

Puck uses the segmentation algorithm described in the paper [Felzenszwalb and Huttenlocher(2004): Efficient Graph-Based Image Segmentation](http://cs.brown.edu/~pff/papers/seg-ijcv.pdf); you can find the source code at [http://cs.brown.edu/~pff/segment/](http://cs.brown.edu/~pff/segment/).
